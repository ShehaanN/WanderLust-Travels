## üöÄ Getting Started

### Installation

1. **Clone the repository** (or download the project):

   ```bash
   git clone https://github.com/ShehaanN/WanderLust-Travels.git
   cd WanderLust-Travels
   ```

2. **Install dependencies**:

   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server**:

   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **Open your browser** and navigate to:

   ```
   http://localhost:3000
   ```
